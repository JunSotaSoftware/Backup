Changes in Ver.1.20b
--------------------
-- Fixed a bug that Backup pattern comments cannot be displayed 
   as chips in Windows 11.

-- Fixed a problem in which "Backup source" was displayed 
   on the screen for selecting a destination folder or a 
   folder not to be backed up.

Changes in Ver.1.20a
--------------------
-- Fixed a bug that referred to uninitialized variables.

Changes in Ver.1.20
--------------------
-- The option to copy files without encryption has been 
   added, even if the source files for backup are encrypted 
   with EFS and the backup destination cannot be encrypted.
