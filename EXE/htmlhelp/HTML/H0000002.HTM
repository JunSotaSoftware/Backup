Backup  Ver1.16
(動作確認環境 Windows 10, Windows 8.1, Windows 7, Windows Vista)


ファイルのバックアップを行うツールです。

以下のような処理を行います。

(1)バックアップ元にないフォルダ、ファイルをバックアップ先から削除します。

(2)バックアップ元にしかないファイル、バックアップ元のほうが日付が新しいファイルを、バックアップ先にコピーします。

バックアップ元のファイルに対しては、変更、削除は一切行いません。
バックアップの基本的な考え方の項目も参考にしてください。

特徴

☆バックアップ元と同じフォルダ・ファイル構成のまま、バックアップ先にコピーします。

☆ファイルの日付をチェックして更新されたファイルのみをバックアップするので、２回目からはバックアップに必要な時間が短縮されます。

☆ドイツ語や韓国語などの多国語文字を、ファイル名に使用したファイルのバックアップに対応しています。

動作確認環境

Backupは以下の環境で動作を確認しています。
  Windows 10 (64bit)
  Windows 8.1 (64bit)
  Windows 7 (64bit)
  Windows Vista (32bit)

このバージョンは、Windows XP, Windows Me, Windows 98, Windows 95では動作しません。
