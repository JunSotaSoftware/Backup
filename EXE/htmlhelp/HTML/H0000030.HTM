Backup Ver0.3 以降のバージョンは、Backup Ver0.22 以前のバージョンとはバックアップパターンの設定方法が異なっています。
以前の設定のままですと転送先が違う場所になってしまいますので、再設定を行ってください。

●バックアップ元がルートディレクトリ（C:\、D:\ など）の場合。

設定を変更する必要はありません。

●バックアップ元がルートディレクトリ以外（C:\Home、D:\Prog\Source など）の場合

Backup Ver0.3 以降では、バックアップ元のフォルダと同じ名前のフォルダを作成し、その中にファイルをコピーします。

例１) バックアップ元が C:\Prog\Source、バックアップ先が D:\ に設定されている場合

◎Backup Ver0.22 以前
D:\ の中にファイルをコピーします。

◎Backup Ver0.3 以降
D:\Source の中にファイルをコピーします。（バックアップ元の C:\Prog\Source の最後の部分である Source と同じ名前のフォルダを作成します)

例２) バックアップ元が C:\Prog\Source、バックアップ先が D:\Backup に設定されている場合

◎Backup Ver0.22 以前
D:\Backup の中にファイルをコピーします。

◎Backup Ver0.3 以降
D:\Backup\Source の中にファイルをコピーします。
